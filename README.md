# Monsoon Empire

A small browser strategy/trading game inspired by Oregon Trail-style travel, old map graphics, and simplified grand strategy mechanics.

You start in London with one ship. Sail across the Indian Ocean, buy goods, sell them back home, build ships, fight rivals, and capture ports.

## How to Play

1. Open `index.html` in a browser.
2. Start in London.
3. Click ports on the map.
4. Sail to ports that sell cheap goods.
5. Buy goods like pepper, cinnamon, tea, silk, porcelain, cotton, nutmeg, and opium.
6. Return to London or another high-price market to sell.
7. Build more ships, repair your fleet, and attack ports when strong enough.
8. End each month to let prices, rivals, and events update.

## Win Conditions

You win if either:

- You earn £50,000, or
- England controls 6 ports.

## Uploading to GitHub Pages

1. Create a new GitHub repository.
2. Upload these files:
   - `index.html`
   - `style.css`
   - `game.js`
   - `README.md`
3. Go to the repository settings.
4. Open **Pages**.
5. Set source to the main branch and root folder.
6. Open the GitHub Pages link.

## Files

- `index.html` — page structure
- `style.css` — old map and parchment styling
- `game.js` — game logic, ports, trading, AI rivals, battles, ships
