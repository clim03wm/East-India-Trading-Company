# Monsoon Empire: Charter Capital

A browser-based idle capitalist trading game for GitHub Pages.

You build ships and choose their trade destinations. Ships automatically sail from London to the selected port, load goods, return to London, earn profit, and repeat.

## Files

- `index.html`
- `style.css`
- `game.js`
- `map.png`

## How to play

1. Open `index.html` or host the repo with GitHub Pages.
2. Select a port on the map.
3. Build ships.
4. Assign idle ships to the selected port.
5. Ships sail automatically and produce profit.
6. Buy upgrades to increase company income.

## GitHub Pages

Upload all files to the root of your repository, then go to Settings → Pages and deploy from the `main` branch, `/root` folder.
