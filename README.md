# Monsoon Empire: Charter Capital

A browser-based idle trading-company game for GitHub Pages.

## How to launch locally

Double-click `index.html`.

## How to publish on GitHub Pages

1. Upload these files to the root of your GitHub repository:
   - `index.html`
   - `style.css`
   - `game.js`
   - `map.png`
   - `README.md`
2. Commit the changes.
3. Go to Settings → Pages.
4. Set Source to `Deploy from a branch`.
5. Set Branch to `main` and Folder to `/root`.
6. Open the GitHub Pages link after it finishes deploying.

## Controls

- Click a port on the map.
- Build a fleet.
- Select a fleet in the Fleet Ledger.
- Assign the selected fleet to a route.
- Fleets sail automatically along the real trade route path.
- Use Pause Time to freeze the clock while still choosing ports and assigning fleets.
